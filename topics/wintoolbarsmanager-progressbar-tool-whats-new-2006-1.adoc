= ProgressBar Tool

With the new  pick:[win-forms="link:infragistics4.win.ultrawintoolbars.v{ProductVersion}~infragistics.win.ultrawintoolbars.progressbartool.html[ProgressBarTool]"]  class, you can now display a progress bar within a toolbar or menu. This tool exposes several properties for controlling the appearance and style of the progress area. Although there is no edit mode for the ProgressBar tool, you can still click and double-click on the progress bar, and the  pick:[win-forms="link:infragistics4.win.ultrawintoolbars.v{ProductVersion}~infragistics.win.ultrawintoolbars.toolbase~toolclick_ev.html[ToolClick]"]  and  pick:[win-forms="link:infragistics4.win.ultrawintoolbars.v{ProductVersion}~infragistics.win.ultrawintoolbars.toolbase~tooldoubleclick_ev.html[ToolDoubleClick]"]  events will fire, respectively.
