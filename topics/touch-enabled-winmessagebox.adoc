= Touch-enabled WinMessageBox

== Topic Overview

=== Purpose

This topic illustrates the Infragistics touch-enabled link:{ApiPlatform}win.v{ProductVersion}~infragistics.win.ultramessagebox.messageboxcontrol_members.html[WinMessageBox]™ control elements.

== Touchable  _WinMessageBox_   Elements

=== Touchable element

Appearing slightly larger in touch-enabled mode, the following screenshots illustrate the  _WinMessageBox_   control’s touchable elements.

.Note:
[NOTE]
====
The control caches and uses state of the touch-enabled setting of the link:{ApiPlatform}win.v{ProductVersion}~infragistics.win.touch.ultratouchprovider_members.html[UltraTouchProvider] for the duration of the message box display. Toggling the `Enabled` property of the link:{ApiPlatform}win.v{ProductVersion}~infragistics.win.touch.ultratouchprovider_members.html[UltraTouchProvider] has no effect on already open message boxes.
====
