= Value Pane

The Value pane lets you specify the start point and the end point of the range on the scale.

pick:[win-forms="link:infragistics4.win.ultrawingauge.v{ProductVersion}~infragistics.ultragauge.resources.gaugerange~startvalue.html[Start]"]  -- Set this value to an integer from 0 to 360. This value sets the start point of the range on the scale.

pick:[win-forms="link:infragistics4.win.ultrawingauge.v{ProductVersion}~infragistics.ultragauge.resources.gaugerange~endvalue.html[End]"]  -- Set this value to an integer from 0 to 360. This value sets the end point of the range on the scale.
