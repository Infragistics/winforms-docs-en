= Visual Studio .NET 2005-Style Docking

A new LayeredWindowWithIndicators value off the  pick:[win-forms="link:infragistics4.win.ultrawindock.v{ProductVersion}~infragistics.win.ultrawindock.dragwindowstyle.html[DragWindowStyle]"]  Enumeration lets you enable the WinDockManager™ to display Visual Studio .NET 2005-style Docking Indicators when you drag a dockable pane or group. While dragging, if the mouse hovers over a docking indicator (e.g., top dock area), a semi-transparent window appears. When you click the docking indicator, the dragged pane or group is docked at the location corresponding to the docking indicator that you clicked.
